# Eva Reed Müzik Portföyü Web Sitesi

Bu proje, AI (Midjourney) tarafından oluşturulmuş yetenekli müzisyen Eva Reed'in çalışmalarını sergilemek için ChatGPT yardımıyla tasarlanmış bir web sitesidir. Responsive tasarımı sayesinde tüm cihazlarda sorunsuz bir kullanıcı deneyimi sunar. HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

## Proje Hakkında

Eva Reed'in müzik portföyü web sitesi, sanatçının albümleri, tur tarihleri ve hakkında bilgiler gibi içerikleri kullanıcılarına sunar. Ayrıca, ziyaretçilerin sanatçının şarkılarından kısa kesitleri dinlemelerine olanak tanıyan bir özellik içerir.

## Başlangıç

Projeyi lokal olarak çalıştırmak ve geliştirmeye başlamak için aşağıdaki adımları takip edin.

### Gereklilikler

Bu projeyi çalıştırmak için herhangi bir özel yazılıma ihtiyaç yoktur. Ancak, dosyaları düzenlemek için bir metin editörüne ve HTML/CSS/JavaScript kodlarını görüntülemek için bir web tarayıcısına ihtiyacınız olacak.

### Kurulum

Bu proje herhangi bir server-side işleme ihtiyaç duymadan statik dosyalar olarak sunulmaktadır.
